Introducing Pico.

A stupidly simple & blazing fast, flat file CMS.
Making the web easy.

Stupidly Simple

Pico makes creating and maintaining a website is as simple as editing text files.

Blazing Fast

Pico is seriously lightweight and doesn't use a database, making it super fast.

No Database

Pico is a "flat file" CMS, meaning no database woe's, no MySQL queries, nothing.

Markdown Formatting

Edit your website in your favorite text editor using simple Markdown formatting.

Twig Templates

Pico use the Twig templating engine, for powerful and flexible themes.

Open Source

Pico is completely free and open source, released under the MIT license.

Want to Contribute?

Help make Pico better by checking out Github repository and submitting pull requests.

If you find a bug please report it on the issues page.
